我会持续更新hashtable，本项目在deepin os下可以良好运行，可以实现一般的查找功能

#feature
*链地址解决冲突
*可以实现基本的查找，插入
*跨平台运行

#install
拷贝一份代码到你的项目下，包含头文件```hash.h```
